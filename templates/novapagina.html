O que é KNN?

O K-Nearest Neighbors é um algoritmo simples usado para classificar dados baseado nos seus 'vizinhos' mais próximos em um espaço de features.

Dataset Íris

Um dataset clássico em machine learning, contendo medições de 3 espécies de flores Íris, ideal para problemas de classificação.

Sobre o Projeto

Esta aplicação demonstra o uso de diferentes modelos de machine learning para classificar flores do dataset Íris. Desenvolvido como atividade prática.
